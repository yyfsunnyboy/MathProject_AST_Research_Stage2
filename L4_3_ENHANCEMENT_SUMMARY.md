## 🔍 L4.3 检测规则

### 1. 零系数项 (严重违规) → 直接 0 分

检测模式：
- `\b0[a-zA-Z]` - 例如 `0x`, `0y`, `0t`（直接相连）
- `\b0\s+[a-zA-Z]` - 例如 `0 x`, `0 y`（有空格）

**处理**：发现一次直接扣100分（即直接 0 分），无法继续得分。

**示例**：
```
错误: 0x^3 + 2x + 1  → 0 分
正确: 2x + 1         → 继续评分
```

---

### 2. 符号未简化 (中度违规) → 每个扣 5 分

检测模式：
- `\+\s*-` - 例如 `5 + -3`（正负符号未合并）
- `-\s*-` - 例如 `x - -5`（负号未合并）
- `\^\s*1\b` - 例如 `x^1`（指数为1应省略，应写成 `x`）

**处理**：每发现一个实例扣5分。

**示例**：
```
错误: 5 + -3       → 扣5分，得5分
错误: x^1 + 2x     → 扣5分，得5分
错误: x - -5       → 扣5分，得5分
```

---

### 3. 冗余系数 (轻微违规) → 每个扣 2 分

检测模式：
- `(?<!\d)\b1[a-zA-Z]` - 例如 `1x`, `1y`（应写成 `x`, `y`）
- `(?<!\d)-1[a-zA-Z]` - 例如 `-1x`, `-1y`（应写成 `-x`, `-y`）

**处理**：每发现一个实例扣2分。

**示例**：
```
错误: 导数是 1x + 3      → 发现 2个 (1x 和 1x) → 扣4分，得6分
错误: 结果是 -1x         → 发现 2个 (-1x 匹配两次) → 扣4分，得6分
错误: 1x + -1y + 2       → 发现 3个 (1x, -1y匹配) → 扣6分 + 中度5分 → 0分
```
