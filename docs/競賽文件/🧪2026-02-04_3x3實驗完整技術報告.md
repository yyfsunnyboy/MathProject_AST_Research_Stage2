### 1. MASTER_SPEC 淨化的驚人效果

**背景**：
- 14b 模型 Token 上限有限（8K-32K）
- 原始 MASTER_SPEC 包含大量實作細節（5,356 tokens）
- 擔心「誤導」模型照抄實作步驟

**淨化策略（V47.14）**：
```python
@staticmethod
def _clean_master_spec(master_spec: str) -> str:
    """移除實作細節，只保留約束"""
    spec_dict = yaml.safe_load(master_spec)
    keys_to_remove = [
        'construction',              # 逐步實作指引
        'implementation_checklist',  # 檢查清單
        'formatting',                # 格式範例
        'variables'                  # 變數定義
    ]
    # 從所有 templates 中移除
    if 'templates' in spec_dict:
        for template in spec_dict['templates']:
            for key in keys_to_remove:
                if key in template:
                    del template[key]
    return yaml.dump(spec_dict, allow_unicode=True, sort_keys=False)
```

```python
        # ✅ 成功則返回
        return {

            'correct_answer': a,
            'answer': a,
            'mode': 1
        }
```

### V47.14: MASTER_SPEC 淨化（2026-02-03）

**背景**：
- 14b 模型 Token 上限限制
- 原始 MASTER_SPEC 過於冗長（5,356 tokens）
- 擔心實作細節誤導模型

**實現**：
```python
# core/prompts/prompt_builder.py

@staticmethod
def _clean_master_spec(master_spec: str) -> str:
    """
    淨化 MASTER_SPEC：移除實作細節，只保留約束
    
    移除欄位：
    - construction: 逐步實作指引（會誤導模型照抄）
    - implementation_checklist: 檢查清單（模型會當成範本）
    - formatting: 格式範例（已在 Domain Helper 中提供）
    - variables: 變數定義（重複資訊）
    
    保留欄位：
    - entities: 實體定義
    - constraints: 約束條件
    - operators: 運算規則
    - templates.complexity_requirements: 複雜度要求
    """
    import yaml
    
    spec_dict = yaml.safe_load(master_spec)
    keys_to_remove = ['construction', 'implementation_checklist', 'formatting', 'variables']
    
    # 從所有 templates 中移除
    if 'templates' in spec_dict:
        for template in spec_dict['templates']:
            for key in keys_to_remove:
                if key in template:
                    del template[key]
    
    return yaml.dump(spec_dict, allow_unicode=True, sort_keys=False)
```

**效果**：
- Token 減少：5,356 → 1,592（**-70.3%**）
- 驗證通過率：100% → 100%（維持）
- 品質分數：7/7 → 7/7（維持）

**學術價值**：
