## 🐛 具體問題

### **問題 1: Ab1 不應該有工具庫**

**位置**: `core/code_generator.py` Line 527-534

**錯誤代碼**:
```python
# 組裝 final_code
if ablation_id >= 2:
    skeleton = build_calculation_skeleton(skill_id)
    final_code = skeleton + "\n" + clean_code
else:
    # Ab1 (Bare) 這裡也加上 skeleton 確保可運行 ← ❌ 錯誤！
    skeleton = build_calculation_skeleton(skill_id)
    final_code = skeleton + "\n" + clean_code  # ← Ab1 也加了工具庫！
```

**問題**:
- Ab1 被注入了 `PERFECT_UTILS`（包含 `fmt_num`, `ensure_int` 等工具函數）
- Ab1 被注入了 `Domain Functions`（例如導數相關函數）
- **破壞了實驗設計的變因分離**：無法區分是模型本身的能力還是工具庫的幫助

**影響**:
- ❌ Ab1 vs Ab2 的對比無效（都有工具庫）
- ❌ 無法測試模型的原生智商
- ❌ 實驗數據失去學術價值

---

### **問題 2: Ab2 不應該有 Healer**

**位置**: `core/code_generator.py` Line 465-466

**錯誤代碼**:
```python
use_regex_healer = (ablation_id >= 2)  # ← ❌ Ab2 和 Ab3 都啟用！
use_ast_healer = (ablation_id >= 3)    # ← 只有 Ab3
```

**問題**:
- Ab2 啟用了 **Regex Healer**（9 個修復項目）
- **破壞了實驗設計的變因分離**：
  - Ab2 vs Ab3 應該只測試 **Healer 的效果**
  - 但 Ab2 已經有 Regex Healer，只差 AST Healer

**影響**:
- ❌ Ab2 vs Ab3 的對比不純粹（Ab2 已經有部分 Healer）
- ❌ 無法準確測量 Healer 的完整貢獻
- ❌ 混淆了 Prompt 工程和 Healer 的效果

## 🔧 修復方案

### **修復 1: Ab1 不注入工具庫**

**修復後代碼**:
```python
# 組裝 final_code
# [2026-02-01 Bug Fix] 嚴格遵守實驗設計：
# - Ab1 (Bare): 無工具庫，純 AI 生成
# - Ab2/Ab3: 有工具庫 (PERFECT_UTILS + Domain Functions)
if ablation_id >= 2:
    # Ab2, Ab3: 注入完整工具庫
    skeleton = build_calculation_skeleton(skill_id)
    final_code = skeleton + "\n" + clean_code
else:
    # Ab1 (Bare): 不注入工具庫，測試模型原生能力
    # 只包含 AI 生成的代碼（靠 random/math 等標準庫）
    final_code = clean_code  # ← 不加 skeleton！
```

**效果**:
- ✅ Ab1 只有純 AI 生成的代碼
- ✅ Ab1 vs Ab2 可以測試工具庫的貢獻
- ✅ 符合實驗設計的"Bare"定義

### **修復 2: Ab2 不啟用 Healer**

**修復後代碼**:
```python
# Ablation Settings
# [2026-02-01 Bug Fix] 嚴格遵守實驗設計的變因分離：
# - Ab1: 無 Healer（僅基礎清理）- 測試模型原生智商
# - Ab2: 無 Healer（僅基礎清理）- 測試 Prompt 工程 + 工具注入
# - Ab3: 有 Healer (Regex + AST) - 測試自癒機制
ablation_config = AblationSetting.query.get(ablation_id)
if ablation_config:
    use_regex_healer = ablation_config.use_regex
    use_ast_healer = ablation_config.use_ast
    ablation_name = ablation_config.name
else:
    # Fallback: 確保變因分離
    use_regex_healer = (ablation_id >= 3)  # 只有 Ab3 啟用 Regex Healer
    use_ast_healer = (ablation_id >= 3)    # 只有 Ab3 啟用 AST Healer
    ablation_name = f"Ablation-{ablation_id}"
```

**效果**:
- ✅ Ab1, Ab2 都不啟用 Healer
- ✅ 只有 Ab3 啟用完整 Healer
- ✅ Ab2 vs Ab3 可以純粹測試 Healer 的效果
