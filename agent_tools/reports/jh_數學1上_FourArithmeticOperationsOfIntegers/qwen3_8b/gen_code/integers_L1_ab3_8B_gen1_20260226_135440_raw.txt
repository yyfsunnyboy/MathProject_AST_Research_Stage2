# 數學題目生成器 - 四則運算（整數）

## 題目範例
計算 $$(-12) \times 5$$ 的值。

## 題目生成邏輯
- **Level 1 (兩項)**: 範圍 -20 ~ 20，僅包含乘法和除法
- **Level 2 (三項混合)**: 範圍 -50 ~ 50，包含乘除混合運算
- **Level 3 (四項混合)**: 範圍 -100 ~ 100，包含加減乘除混合運算

## 題目格式
- 乘號用 `\times`，除號用 `\div`
- 最後的算式字串【必須】用雙錢號 `$$...$$` 包起來

## 生成規則
- 使用標準 Python 運算生成數值（+、-、*、//）
- 絕對禁止使用 `eval` 或 `safe_eval` 處理 `question_text` 字串求值
- 數學計算與字串排版分離：必須先完成所有整數純數字計算，最後組裝文字時再呼叫 `IntegerOps.fmt_num(n)`
- 除法運算時，必須確保整除
