# 數學題目生成器（整數四則運算）

## 題目生成邏輯說明
- 題目會根據難度層級（Level 1, Level 2, Level 3）生成不同範圍的整數運算題
- 會自動處理負數格式，並使用 LaTeX 格式顯示運算式
- 會確保運算結果為整數，並符合整除要求
- 會使用 `\times` 表示乘法，`\div` 表示除法

## 生成的題目範例
### Level 1（兩項運算）
- 題目：計算 $$( -12 ) \times ( 5 )$$ 的值。
- 答案：-60

### Level 2（三項混合運算）
- 題目：計算 $$( 45 ) \div ( -9 ) \times ( -6 )$$ 的值。
- 答案：30

### Level 3（四項混合運算）
- 題目：計算 $$( -18 ) \times ( 7 ) + ( -60 ) \div ( 10 )$$ 的值。
- 答案：-126

## 生成的題目結構
```python
{
    'question_text': '計算 $$( -12 ) \times ( 5 )$$ 的值。',
    'answer': '',
    'correct_answer': '-60',
    'mode': 1
}
```

## 運算邏輯說明
- 會先生成運算數值，再組裝運算式
- 會確保除法運算結果為整數
- 會使用 `IntegerOps.fmt_num(n)` 格式化負數
- 會使用 `$$...$$` 包裝運算式

## 題目生成範例
```python
{
    'question_text': '計算 $$( -18 ) \times ( 7 ) + ( -60 ) \div ( 10 )$$ 的值。',
    'answer': '',
    'correct_answer': '-126',
    'mode': 1
}
```

## 題目生成規則
| 難度層級 | 運算數值範圍 | 運算符數量 | 運算符種類 | 除法限制 |
|----------|--------------|------------|------------|----------|
| Level 1  | -20 ~ 20     | 2          | 乘法、除法 | 商數 ≤ 10 |
| Level 2  | -50 ~ 50     | 3          | 乘法、除法 | 商數 ≤ 20 |
| Level 3  | -100 ~ 100   | 4          | 乘法、除法 | 商數 ≤ 30 |
