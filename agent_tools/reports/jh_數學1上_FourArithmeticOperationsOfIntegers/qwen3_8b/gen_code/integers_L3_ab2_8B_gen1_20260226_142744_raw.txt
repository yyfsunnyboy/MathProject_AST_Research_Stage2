# 數學題目生成器（整數四則運算）

## 題目生成邏輯說明
- 題目會根據難度層級（Level 1, Level 2, Level 3）生成不同範圍的整數運算題
- 題目會自動處理負數格式，並使用 LaTeX 語法顯示運算式
- 會確保所有運算都符合整除要求，避免小數
- 會自動生成正確答案

## 生成的題目範例
### Level 1（兩項運算）
- 計算 $$(12) \times (-5)$$ 的值。
- 計算 $$(-24) \div (-6)$$ 的值。

### Level 2（三項混合運算）
- 計算 $$(-48) \div (-6) \times (7)$$ 的值。
- 計算 $$(15) \times (-10) \div (-5)$$ 的值。

### Level 3（四項混合運算）
- 計算 $$(12) \times (-8) + (-60) \div (-10)$$ 的值。
- 計算 $$(20) - (-40) \div (-8) \times (5)$$ 的值。

## 題目生成器使用說明
1. 調用 `generate(level)` 函數，指定難度層級（1, 2, 3）
2. 會自動生成符合要求的運算式與正確答案
3. 使用 `check(user_answer, correct_answer)` 函數驗證答案

## 注意事項
- 所有運算都必須使用標準 Python 運算符（+、-、*、//）
- 除法運算時會先計算商數與除數，再反推被除數以確保整除
- 題目中的乘號使用 `\times`，除號使用 `\div`
- 最後的算式字串必須用雙錢號 `$$...$$` 包起來

## 生成的題目格式
```python
{
    'question_text': '計算 $$(12) \times (-5)$$ 的值。',
    'answer': '',
    'correct_answer': '60',
    'mode': 1
}
```

## 答案驗證
```python
check('60', '60')  # 返回 {'correct': True, 'result': '正確'}
check('60.0', '60')  # 返回 {'correct': True, 'result': '正確'}
check('59', '60')  # 返回 {'correct': False, 'result': '錯誤'}
```
