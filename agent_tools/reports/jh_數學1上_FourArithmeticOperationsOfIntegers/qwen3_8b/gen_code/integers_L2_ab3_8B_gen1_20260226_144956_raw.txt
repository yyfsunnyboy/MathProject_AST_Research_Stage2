# 數學題目生成器（整數四則運算）

## 產生的題目格式
- 題目以 `$$...$$` 包裹
- 乘法用 `\times`，除法用 `\div`
- 答案為整數
- 負數自動加括號

## 產生規則
- Level 1: 兩項運算，範圍 -20 ~ 20
- Level 2: 三項混合運算，範圍 -50 ~ 50
- Level 3: 四項混合運算，範圍 -100 ~ 100
- 絕對禁止使用 `eval` 或 `safe_eval` 處理 `question_text` 字串求值
- 數學計算與字串排版分離
