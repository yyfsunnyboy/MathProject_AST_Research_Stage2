# 數學題目生成器（整數四則運算）

## 題目生成邏輯說明
- 題目會根據難度層級（Level 1, Level 2, Level 3）生成不同範圍的整數運算題
- 題目會自動處理負數的格式（自動加括號）
- 會使用標準 Python 運算生成數值（+、-、*、//），並確保除法先決定商數與除數再反推被除數以達到整除
- 會使用 `\times` 表示乘法，`\div` 表示除法
- 最後的算式字串【必須】用雙錢號 `$$...$$` 包起來

## 生成的題目範例
### Level 1（兩項運算）
- 計算 $$(-12) \times (5)$$ 的值。
- 計算 $$(18) \div (-3)$$ 的值。

### Level 2（三項混合運算）
- 計算 $$(36) \div (-6) \times (7)$$ 的值。
- 計算 $$(20) \times (-5) \div (-10)$$ 的值。

### Level 3（四項混合運算）
- 計算 $$(-8) \times (4) + (-30) \div (-5)$$ 的值。
- 計算 $$(15) - (-20) \div (-4) \times (3)$$ 的值。
