## 🎓 實驗設計支持

### Ablation Study (消融研究)
```
Ab1 (Bare):       簡單 Prompt + 無修復   → 測試模型基線
Ab2 (Engineered): 精細 Prompt + 無修復   → 測試提示工程貢獻
Ab3 (Full-Healing): 精細 Prompt + Healer → 測試系統完整能力
```

### 模型對比
```
Gemini (Cloud):   高準確度、高成本
Qwen 14B (Local): 中等質量、無網路依賴
Qwen 7B (Local):  快速推理、低資源消耗
```

### 統計維度
```
全局級: 整體成功率、Token 消耗、執行時間
模型級: 每個模型的表現對比
技能級: 不同技能的難度差異
```
